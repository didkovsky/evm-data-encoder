# EVM Calldata Encoder (frontend)

Lightweight frontend utility to **encode EVM smart contract calldata** from a function signature + parameter values. Runs fully in the browser (no RPC).

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

- Push to `main`
- In GitHub repo settings: **Pages → Build and deployment → Source = GitHub Actions**
- The included workflow (`.github/workflows/deploy.yml`) will build and deploy automatically.

## Notes on inputs

- **Method signature** example: `approve(address,uint256)`
- **Arrays**: provide JSON, e.g. `[1,2,3]` or `["0x...","0x..."]`
- **bool**: `true` / `false`
